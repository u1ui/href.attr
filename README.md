# [u1-href] - attribute
Every element can be a link

## Ussage

```html
<div u1-href="http://schwups.ch" u1-href-target=_blank>
    Pseudo link
</div>
```

## Note

Always offer an A-Tag with the same link. Screen readers do not recognize that this is a link!
