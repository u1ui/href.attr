# u1-href
Turns every element in a link

# Ussage

```html
<table>
    <tr u1-href="detail.html">
        <td> John
        <td> Muller
        <td> <a href="detail.html">Show me the details</a>
    <tr>
</table>
```


# Note
Always offer an A-Tag with the same link. Screen readers do not recognize that this is a link!
